## 📋 نظرة عامة

مشاريع **GNUTUX** هي مجموعة من الحلول البرمجية مفتوحة المصدر التي تهدف إلى توفير أدوات احترافية ومجانية للمستخدمين والمطورين العرب. يجمع هذا المستودع بين التصميم العصري والوظائف المتقدمة لتقديم تجربة مستخدم استثنائية.

### ✨ المميزات الرئيسية للمنصة

- 🌐 **دعم ثنائي اللغة** - العربية والإنجليزية مع تبديل سلس
- 🌙 **الوضع الليلي** - تجربة مريحة للعين في جميع الأوقات
- 📱 **تصميم متجاوب** - يعمل على جميع الأجهزة والمتصفحات
- 🔍 **عرض تفاعلي** - بطاقات مشاريع مع شعاراتها الخاصة
- 📖 **ملفات توضيحية** - عرض README لكل مشروع في نافذة موسعة
- 🔗 **روابط خارجية** - جميع الروابط تفتح في تبويب جديد للأمان

## 🚀 التشغيل السريع

### متطلبات النظام
```bash
- متصفح حديث (Chrome, Firefox, Safari, Edge)
- اتصال بالإنترنت
- خادم ويب محلي (للتطوير فقط)
```

### تثبيت المشروع محلياً

```bash
# استنساخ المستودع
git clone https://github.com/SalehGNUTUX/SalehGNUTUX.github.io.git

# الدخول إلى المجلد
cd SalehGNUTUX.github.io

# تشغيل خادم محلي (باستخدام Python 3)
python -m http.server 8000

# أو باستخدام Node.js
npx serve .
```

ثم افتح المتصفح على: `http://localhost:8000`
